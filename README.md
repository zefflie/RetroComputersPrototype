# Как запустить
В проекте Workspace есть папка Examples с примерами кода на няссемблере.<br>
Для каждого файла есть свой ps1 скрипт, что запускает сборку компилятора и эмулятора, а после компилирует и запускает указанный исходный файл.<br>
Запускать эти скрипты нужно из той-же папки Examples.

# CLI компилятора
```
nyassembler.exe [-o <конечный файл>] <... исходные файлы>
```

В данный момент няссемблер не способен компилировать несколько исходных файлов в одну программу, поэтому обрабатывает только первый указанный.

# CLI эмулятора
```
emulator.exe [-f <исполняемый файл>]
```

Исполняемый файл размещается в начале ОЗУ ввиде ROM памяти, попытки записи в эту область приведут к ошибке эмулятора. Диапазон ROM памяти равен размеру файла.

# Характеристики эмулятора
- 64кб оперативной памяти
- i8080 центральный процессор
- Вымышленный текстовый видеоадаптер

Процессор не имеет ограничений по скорости выполнения, поэтому может быть гораздо производительнее чем в действительности.<br>
Текстовый видеоадаптер выводится в консоль, обладает разрешением 64x32 знаков. Кодировка знаков зависит от консоли. Использует адресный диапазон ОЗУ 0xB000-0xB800 в качестве буфера, который и выводится в консоль.
